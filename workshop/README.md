# Workshop Files

This folder contains the complete Google Apps Script code used in the main tutorial.

## Files

- **`Code.gs`** - Complete Google Apps Script implementation for the Bolna AI + Google Sheets integration

## Usage

Copy the contents of `Code.gs` into your Google Apps Script project as described in the [main tutorial](../README.md).

Make sure to update the configuration variables at the top of the file with your actual Bolna API credentials.
